Projeto Final

Neste projeto, vocês deverão desenvolver um aplicativo mobile completo utilizando React Native, aplicando de forma integrada os principais conceitos e tecnologias estudados ao longo da disciplina.

O projeto final poderá ser uma evolução dos trabalhos desenvolvidos anteriormente, ampliando o escopo e adicionando novas funcionalidades. Entretanto, caso o projeto atual não permita a aplicação adequada dos requisitos mínimos descritos abaixo, será necessário redefinir ou recriar o projeto com um escopo maior, garantindo que todos os conteúdos da disciplina sejam contemplados.
Requisitos obrigatórios

O aplicativo deverá utilizar obrigatoriamente os seguintes recursos e conceitos:

    Interface mobile estruturada e organizada;

    Navegação entre telas utilizando React Navigation;

    Uso adequado de componentes e gerenciamento de estado;

    Implementação de listas performáticas utilizando FlatList ou SectionList;

    Consumo de APIs externas e tratamento de requisições assíncronas;

    Persistência local simples utilizando AsyncStorage;

    Persistência de dados utilizando SQLite;

    Utilização da câmera do dispositivo para captura de imagens;

    Integração com GPS e mapas, utilizando recursos de geolocalização;

    Tratamento adequado de erros e validações;

    Aplicação de boas práticas de organização e arquitetura do projeto.

Observações importantes

    O projeto deve demonstrar claramente a aplicação prática dos conteúdos estudados durante a disciplina.

    Não será aceito um projeto que implemente apenas parte significativa dos requisitos obrigatórios.

    Caso o escopo inicial seja pequeno e não comporte a utilização de todos os recursos exigidos, você deverá expandir ou redefinir o projeto para atender integralmente aos requisitos.
